## 【職務要約/自己PR】
新卒にて中小ソフトウェア企業にて正社員として5年間、受託開発や客先開発にてJavaやOracleを使ったシステム開発に携わりました。  

その後フリーランスに転身し、Railsを主に使ったバックエンドのWEB開発をメインとしながら、NuxtやJavascriptなどのフロントエンドや、DBのパフォーマンスチューニング、terraformを使ったAWSインフラ構築などをフルスタックに対応しながら各プロジェクトにて従事してきました。  

その中で、スクラムを使ったアジャイル開発を直近では主に行なっており、エンジニアだけでなくスクラムでのプロジェクト貢献ができるように認定スクラムマスターの資格を取得し、エンジニアチームだけでなくビジネスサイドやデザイナー、QA、他エンジニアチームと連携しながら各プロジェクトのシステム開発に貢献しております。今後も新たなプロジェクト先でもフルスタックで活躍できるようにしていければと考えております。  

また、システム開発以外では、エンジニアの採用面談/募集要項作成や新規チームのチームビルディング、新規エンジニアのオンボーディングといった対応などにも微力ながら尽力しておりました。そのようなチームやプロジェクトへのリードスキルも今後高めていけるよう精進していきたいと考えております、何卒よろしくお願い致します。
